vscode Git 확장자 설치 & 사용법

vscode에서 Git을 사용하기전 Git설치를 먼저 해야한다. Git설치

1.vscode 설치 완료 후, extensions(확장설치)에서 아래 확장팩을 설치.
- GitLens
- Git History

2.Github 이용시 방법.

1) Git 계정정보 입력(terminal실행 : 최초1회 입력) git config --global.user.email "유저 이메일" git config --global.user.name "유저 이름"

3.Repository 생성 및 git 연동완료

  1. 1) 생성된 Github저장소에서 주소를 가져온다.
  2. 2) ctrl+shift+p단축키
  3. 3) Git:Clone입력 후 복사한 링크를 입력
  4. 4) Github와 연동할 폴더 선택 후 레퍼지토리 연동
  5. 5) 해당 폴더에 폴더+파일생성 후 push Repository 클릭(하단 이미지 참고)
